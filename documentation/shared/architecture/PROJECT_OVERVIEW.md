# MatuX STEM 学习平台 - 项目概览

## 项目简介

MatuX 是一个面向学生的 STEM 学习平台，融合人工智能、虚拟实验和游戏化激励，为学生提供沉浸式的编程与 STEM 学习体验。支持桌面端（Electron）和移动端（Flutter）。

> **模块解耦说明**: 课件管理模块已解耦至 OpenMTSciEd 项目，机构管理模块已解耦至 OpenMTEduInst 项目。学生账号在三项目间互联互通。

## 核心价值主张

### 对学生
- **沉浸式学习**：AI编程 + STEM实验 + 游戏化激励
- **个性化学习**：AI教师 + 智能推荐 + 学习画像
- **全平台体验**：桌面端(Electron) + 移动端(Flutter) + Web端

### 对开发者
- **高效编码**：AI代码助手，提高开发效率
- **统一规范**：设计系统保证代码一致性
- **快速部署**：完善的DevOps流程

## 功能模块详解

### 1. AI代码生成服务 (ai_service/)
- **核心技术**：LangChain + 多种大语言模型
- **主要功能**：
  - 代码自动补全和生成
  - 错误检测和修复建议
  - 代码重构优化
  - 多语言支持（Python, JavaScript, Java等）

### 2. 智能推荐系统 (recommendation_service/)
- **算法基础**：协同过滤 + 内容推荐 + 深度学习
- **应用场景**：
  - 个性化学习路径推荐
  - 相关课程内容推荐
  - 学习资源智能匹配

### 3. 电商支付系统 (payment_service/)
- **集成能力**：支付宝、微信、银联等多种支付渠道
- **核心特性**：
  - 完整的订单生命周期管理
  - 实时交易监控
  - 财务对账功能

### 4. 硬件认证服务 (hardware_certification/)
- **物联网支持**：设备注册、认证、管理
- **安全机制**：
  - 设备身份验证
  - 数据传输加密
  - 访问权限控制

### 5. ~~多租户管理系统~~ (已解耦至 OpenMTEduInst)
> 此模块已迁移至 OpenMTEduInst 项目，MatuX 仅保留兼容存根。

### 6. ~~许可证管理系统~~ (已解耦至 OpenMTEduInst)
> 此模块已迁移至 OpenMTEduInst 项目，MatuX 仅保留兼容存根。
  - 按时间周期授权
- **管理功能**：
  - 许可证生成和分发
  - 使用情况监控
  - 自动续期提醒
