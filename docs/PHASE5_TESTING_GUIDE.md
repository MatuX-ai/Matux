## 💻 Electron 测试（可选）

### 问题说明
Electron 二进制文件下载失败，可能是网络问题。

### 解决方案

#### 方案 1: 手动安装 Electron（推荐）

```bash
# 1. 清理失败的安装
Remove-Item -Recurse -Force node_modules\electron

# 2. 设置 Electron 镜像（国内用户）
$env:ELECTRON_MIRROR="https://npmmirror.com/mirrors/electron/"

# 3. 重新安装
npm install --save-dev electron@latest

# 4. 验证安装
npx electron --version

# 5. 启动 Electron 应用
npx electron electron/main.js
```

#### 方案 2: 使用浏览器测试（当前推荐）

由于 Electron 安装问题，建议先使用浏览器测试 Phase 5 功能。

## 🐛 常见问题

### Q1: 看不到首次启动引导？
**A**: 可能已完成首次运行。删除标记文件后刷新页面：
```powershell
Remove-Item "$env:APPDATA\imatuproject-design-system\first-run-completed.flag"
```

### Q2: 推荐区域为空？
**A**: 推荐引擎需要设备评级数据。确保：
1. 设备评估已完成
2. 插件注册表已初始化
3. 控制台无错误信息

### Q3: 点击插件卡片无反应？
**A**: 检查以下几点：
1. 浏览器控制台是否有错误
2. `window.pluginAPI` 是否可用
3. 插件数据是否正确加载

### Q4: 评分评论无法提交？
**A**: 检查：
1. 表单是否填写完整
2. Electron IPC 是否可用（浏览器模式可能受限）
3. 控制台错误信息

### Q5: Electron 无法启动？
**A**: 
1. 检查 Electron 是否安装: `npx electron --version`
2. 检查 main.js 路径: `Test-Path electron\main.js`
3. 查看控制台错误信息
4. 尝试重新安装 Electron
