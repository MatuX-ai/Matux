- 支持无限层级

### 2. 三态复选框算法

**状态判断逻辑**:
```typescript
isModuleChecked(module): boolean {
  return module.children.every(child => isSelected(child.id));
}

isModuleIndeterminate(module): boolean {
  const checkedCount = module.children.filter(child => isSelected(child.id)).length;
  return checkedCount > 0 && checkedCount < module.children.length;
}
```

### 3. 动态筛选算法

**多条件组合筛选**:
```typescript
applyFilter(): void {
  let result = [...this.allPermissions];

  // 模块筛选
  if (this.selectedModule) {
    result = result.filter(p => p.module === this.selectedModule);
  }

  // 搜索筛选
  if (this.searchKeyword) {
    const keyword = this.searchKeyword.toLowerCase();
    result = result.map(module => {
      const matchedChildren = module.children?.filter(child =>
        child.name.toLowerCase().includes(keyword) ||
        child.code.toLowerCase().includes(keyword)
      );
      
      const moduleMatched = module.name.toLowerCase().includes(keyword);
      
      if (moduleMatched) return module;
      else if (matchedChildren?.length > 0) return { ...module, children: matchedChildren };
      return null;
    }).filter((p): p is Permission => p !== null);
  }

  this.filteredPermissions = result;
}
```

### 4. 集合操作优化

**使用 Set 替代 Array**:
```typescript
selectedPermissionIds: Set<number> = new Set();

// O(1) 查找复杂度
isSelected(permissionId: number): boolean {
  return this.selectedPermissionIds.has(permissionId);
}

// O(1) 添加/删除
onPermissionToggle(permissionId: number, event: MatCheckboxChange): void {
  if (event.checked) {
    this.selectedPermissionIds.add(permissionId);
  } else {
    this.selectedPermissionIds.delete(permissionId);
  }
}
```

## 📞 使用说明

### 访问路径

```
/management/organization/{机构 ID}/roles
```

### 操作流程

**创建新角色**:
1. 点击右上角"创建角色"按钮
2. 填写基本信息（名称、代码、描述）
3. 选择数据权限范围
4. 点击"配置权限"按钮
5. 在权限配置对话框中选择权限
   - 可按模块筛选
   - 可搜索权限关键词
   - 可全选/取消全选
   - 可勾选模块级联选择
6. 确认选择后返回表单
7. 点击"创建"提交

**编辑角色**:
1. 找到要编辑的角色卡片
2. 点击"编辑"按钮
3. 修改基本信息或权限配置
4. 点击"保存"提交

**删除角色**:
1. 找到要删除的角色
2. 点击"删除"按钮
3. 确认删除操作
4. 系统角色不可删除（按钮禁用）

**配置权限**:
1. 打开角色编辑对话框
2. 查看已选权限统计
3. 点击"配置权限"按钮
4. 调整权限选择
5. 确认后自动更新统计
