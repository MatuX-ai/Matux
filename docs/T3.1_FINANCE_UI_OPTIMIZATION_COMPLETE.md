## 📞 使用说明

### 访问路径

```
/management/organization/{机构 ID}/finance
```

### 操作流程

**查看财务概览**:
1. 进入页面自动加载统计数据
2. 4 个彩色卡片显示关键指标

**管理学费**:
1. 切换到"学费管理"标签
2. 使用筛选条件查找记录
3. 点击操作按钮执行收款/查看详情/退费

**发放薪酬**:
1. 切换到"薪酬管理"标签
2. 选择月份
3. 点击"批量计算"（如需）
4. 点击"发放"按钮

**设置课程定价**:
1. 切换到"课程定价"标签
2. 查看现有定价卡片
3. 点击"编辑"修改价格
4. 点击"启用/停用"切换状态

**查看财务报表**:
1. 切换到"财务报表"标签
2. 选择月份
3. 查看收入/支出/利润详情
4. 点击"导出报表"
