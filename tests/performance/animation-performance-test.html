测试说明

  • Hero 渐显测试: 检查标题、副标题、按钮的逐行渐显效果（总时长应 < 0.6s）
  • 卡片 Hover 测试: 鼠标悬停在卡片上，检查位移和图标旋转（过渡时间 0.2s）
  • 按钮 Hover 测试: 点击/悬停按钮，检查缩放和阴影效果（响应时间 0.15s）
  • 滚动渐入测试: 向下滚动页面，观察灰色条目的渐入效果（时长 0.4s）
  • 性能监控: 右上角实时显示 FPS 和动画时长
