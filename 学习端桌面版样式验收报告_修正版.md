## ⚠️ 需要优化的部分（视觉风格对标）

### P1 - 重要优化（影响视觉一致性）

#### 1. 渐变色使用不足

**原型视觉风格**：
```css
/* 大量使用渐变色提升视觉层次 */
bg-gradient-to-br from-indigo-500 to-purple-600  /* AI推荐卡片 */
bg-gradient-to-r from-orange-500 to-red-500      /* 每日挑战 */
bg-gradient-to-r from-blue-500 to-blue-600       /* 用户消息气泡 */
```

**当前实现**：
```css
/* 仅在成就徽章使用渐变 */
background: linear-gradient(135deg, var(--color-secondary-50, #eff6ff) 0%, var(--color-secondary-100, #dbeafe) 100%);
```

**差距**：
- 原型在多个关键组件使用渐变色（AI推荐、每日挑战、按钮、消息气泡）
- 当前实现几乎全是纯色，视觉冲击力不足
- 缺少原型的"科技感"和"活力感"

**修改建议**：
```css
/* 添加到全局样式或设计令牌 */
:root {
  --gradient-ai-recommend: linear-gradient(135deg, #6366f1 0%, #9333ea 100%);
  --gradient-daily-challenge: linear-gradient(135deg, #f97316 0%, #ef4444 100%);
  --gradient-primary-btn: linear-gradient(135deg, #3b82f6 0%, #2563eb 100%);
  --gradient-accent: linear-gradient(135deg, #8b5cf6 0%, #6366f1 100%);
}
```

**优先级**：🟠 重要（提升视觉品质感）

#### 2. 卡片阴影效果偏弱

**原型规范**：
```css
shadow-md    /* 0 4px 6px rgba(0,0,0,0.1) */
shadow-lg    /* 0 10px 15px rgba(0,0,0,0.1) */
shadow-xl    /* 0 20px 25px rgba(0,0,0,0.15) */
```

**当前实现**：
```css

```

**差距**：
- 原型阴影更强，层次感更明显
- 当前实现阴影过弱，卡片"浮起"感不足
- 缺少悬浮效果的阴影增强（hover:shadow-xl）

**修改建议**：
```css
.card {

  transition: box-shadow 0.2s, transform 0.2s;
  
  &:hover {
    box-shadow: 0 10px 15px rgba(0,0,0,0.12);
    transform: translateY(-2px);
  }
}
```

**优先级**：🟠 重要（提升视觉层次）

#### 4. 交互动画不足

**原型动画效果**：
- 页面切换动画（framer-motion）
- 按钮点击缩放（whileTap: scale: 0.9）
- 悬浮效果（whileHover: scale: 1.05, y: -4）
- 消息出现动画（opacity: 0 → 1, y: 10 → 0）

**当前实现**：
- 基础 CSS transitions（transform 0.2s）
- 成就徽章悬浮效果
- 缺少点击反馈和出现动画

**修改建议**：
```css
/* 添加 Angular animations 或增强 CSS */
.card {
  transition: transform 0.2s, box-shadow 0.2s;
  
  &:hover {
    transform: translateY(-4px) scale(1.02);
    box-shadow: 0 10px 20px rgba(0,0,0,0.15);
  }
  
  &:active {
    transform: scale(0.98);
  }
}

/* 添加淡入动画 */
@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(10px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

.section {
  animation: fadeInUp 0.3s ease-out;
}
```

**优先级**：🟡 一般（提升用户体验）
