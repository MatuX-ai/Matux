## 一、报告头

### 1.1 审计时间

- **开始时间**：2026-06-13（Saturday）
- **审计方法**：静态分析 + Python WCAG 2.1 相对亮度计算（精确浮点）
- **计算公式**：
  - 相对亮度 L = 0.2126·R' + 0.7152·G' + 0.0722·B'
  - 其中 R' = RsRGB/12.92 (若 RsRGB ≤ 0.03928)，否则 ((RsRGB+0.055)/1.055)^2.4
  - 对比度 = (L₁ + 0.05) / (L₂ + 0.05)（L₁ 为较亮）

### 1.3 审计标准 (WCAG 2.1 AA)

| 元素类型 | 最小对比度 |
|--------|-----------|
| 正文文字（< 18pt / < 14pt 粗体） | **4.5 : 1** |
| 大号文字（≥ 18pt，或 ≥ 14pt 粗体） | **3.0 : 1** |
| UI 组件、图标 | **3.0 : 1** |
| 非装饰性图形 | **3.0 : 1** |
| 装饰性元素 | 不强制 |

字号换算：1pt = 4/3 px ≈ 1.333px；18pt = 24px；14pt = 18.667px

### 1.4 优先级分级（本文档定义）

| 优先级 | 定义 |
|------|----|
| 🔴 **高** | 文字 < 3.0:1 或 图标/UI < 2.0:1（严重不达标） |
| 🟠 **中** | 文字 3.0–4.4:1 或 图标/UI 2.0–2.9:1（轻微不达标） |
| 🟡 **低** | 边界值（依赖主题默认值；或装饰性元素；或轻微越界但不影响主体可读性） |
| ✅ **通过** | 满足标准 |

### 5.1 关键计算公式

```
# WCAG 2.1 相对亮度
L = 0.2126 * R' + 0.7152 * G' + 0.0722 * B'

# sRGB → 线性化
R' = RsRGB / 12.92                          if RsRGB ≤ 0.03928
R' = ((RsRGB + 0.055) / 1.055) ^ 2.4        otherwise

# 对比度 (L1 ≥ L2)
contrast = (L1 + 0.05) / (L2 + 0.05)
```
