## 2. 审计标准与阈值

| 类别 | WCAG 条款 | 阈值 | 说明 |
|------|-----------|------|------|
| **正文文字** | 1.4.3 (AA) | **≥ 4.5:1** | 字号 < 18pt 且非加粗，或 < 14pt |
| **大号文字** | 1.4.3 (AA) | **≥ 3:1** | 字号 ≥ 18pt (24px)，或 ≥ 14pt 加粗 (18.67px) |
| **UI 组件** | 1.4.11 (AA) | **≥ 3:1** | 按钮、图标、输入框边框等 |
| **非装饰性图形** | 1.4.11 (AA) | **≥ 3:1** | 图表线、数据点、状态色块等 |

**优先级分类（本审计自定义）**：

| 优先级 | 文本元素 | 图标/非文本元素 | 影响 |
|--------|----------|----------------|------|
| **🔴 高** | CR < 3:1 | CR < 2:1 | 严重不可读 |
| **🟡 中** | 3:1 ≤ CR < 4.5:1 | 2:1 ≤ CR < 3:1 | AA 失败 |
| **🟢 低** | 4.5:1 ≤ CR < 4.7:1（边缘） | 边界值 | 视觉一致性 |
