```python
class TaskOrchestrationService:
    """联动任务编排服务"""
    
    async def submit_stage1_result(...):
        """提交第一阶段 AI 模型训练结果"""
        # - 保存模型文件
        # - 自动评测准确率
        # - 发放积分奖励
        # - 部署到虚拟实验室
    
    async def submit_stage2_result(...):
        """提交第二阶段硬件集成结果"""
        # - 验证系统集成
        # - 分析系统日志
        # - 评估性能指标
        # - 发放积分奖励
    
    async def get_leaderboard(...):
        """获取任务排行榜"""
```

**关键 API**:
- `submit_stage1_result()` - 阶段 1 提交
- `submit_stage2_result()` - 阶段 2 提交
- `get_leaderboard()` - 排行榜查询
- `_award_stage1_xp()` - 阶段 1 积分计算
- `_evaluate_performance()` - 性能评估

### 2. API 路由（新增 1 个）
**文件**: `backend/routes/linked_task_routes.py`  
**代码量**: 351 行  

**API 端点**:
| 方法 | 路径 | 功能 |
|------|------|------|
| POST | `/{task_id}/stage/1/submit` | 提交阶段 1 |
| POST | `/{task_id}/stage/2/submit` | 提交阶段 2 |
| GET | `/{task_id}/leaderboard` | 获取排行榜 |
| GET | `/{task_id}/my-submissions` | 我的提交 |
| GET | `/{task_id}/details` | 任务详情 |
| POST | `/{task_id}/register` | 报名任务 |

**使用示例**:
```bash
# 提交 AI 模型
curl -X POST \
  http://localhost:8000/api/v1/org/1/ai-edu/linked-tasks/greenhouse_001/stage/1/submit \
  -F "model_file=@plant_health_classifier.pth" \
  -F "training_report=@report.json"

# 查看排行榜
curl http://localhost:8000/api/v1/org/1/ai-edu/linked-tasks/greenhouse_001/leaderboard?limit=10
```
